Shell Script 功能積木組合器

把常用 shell 功能做成積木(function),點選組合、即時預覽,輸出可直接在 Linux 執行的 .sh

🧱 積木庫

✏️ 編輯積木

🧩 組合區

從積木庫按「加入」放入積木;或建立群組,把幾個積木放進去、對整組勾一次 SSH。
尚未加入任何積木。
🖧 SSH 機器清單設定
清單一行一台(可 user@host,# 註解);執行時可用 HOST_LIST=路徑 SSH_PARALLEL=數量 覆寫。需事先佈好 SSH 金鑰 (BatchMode 不會問密碼,連不上直接記失敗)。
⚠️ 並行模式:各積木在子 shell 同時跑,彼此不能共享變數、不保證先後順序; 輸出各自暫存、結束後依加入順序完整印出。有相依關係的積木請用依序模式。
(加入積木後這裡會即時顯示產生的腳本)
